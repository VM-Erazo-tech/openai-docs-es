## 🛠️ Cómo empezar en 30 segundos

Para correr los ejemplos de este repositorio:

1. **Clona el repositorio:**
   ```bash
   git clone https://github.com/VM-Erazo-tech/openai-docs-es.git
   cd openai-docs-es
     ```
2. **Instala la librería oficial:**

    npm install openai
3. **Configura tu API Key:**

   export OPENAI_API_KEY="tu_api_key_aqui"

## 🧠 ¿Qué vas a aprender aquí?

Más que copiar y pegar código, la idea es entender qué está pasando:

- Cómo funciona el ciclo request → response  
- Cómo estructurar una API call correctamente  
- Cómo manejar errores sin romper tu app  
- Cómo usar modelos eficientes como `gpt-4o-mini`
